## What changed

Knowledge bases no longer offer Sitemap as a separate data source type. Legacy v1 sitemap entries are migrated as regular URL entries.

## Why this matters to the user

Users will no longer see a Sitemap/Website source option in the knowledge base UI. Existing v1 sitemap sources remain visible as URL sources after migration.

## What the user should do

Use URL sources for individual web pages. There is no automatic sitemap expansion replacement in v2.
